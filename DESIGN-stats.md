# 能力素質表(玩家屬性與公式)

> 依 game.js v0.13.1 整理。所有數值以程式碼為準,改動時同步更新本表。
> 符號:`lv`=角色等級、`cd.*`=升級卡層數、`meta.up.*`=永久強化等級、`enh`=裝備強化等級、`perkV()`=卡片效果、部位數值皆已含強化倍率 `enhMul = 1 + 0.05×enh`。

## 1. 核心屬性公式

### 攻擊力 atkPow
```
base = 8 + lv×2.5 + 武器攻擊(含強化) + (劍士 +4)
倍率 = (1 + 0.12×卡.atk)        // 力量湧現
     × (1 + 0.04×永久.atk)      // 攻擊強化
     × (1 + 飾品攻擊%)          // 護符 atkMul(含強化)
     × (狂暴中 ? 1.3 : 1)       // 狂暴技能
     × (1 + 0.30×血祭) × (1 + 0.40×蠻力) × (1 + 0.45×玻璃大砲)
     × (HP<35% ? 1 + 0.50×絕地反擊 : 1)
攻擊力 = base × 倍率
```

### 傷害 / 爆擊
```
爆擊率 = 0.08 + 0.06×卡.crit + 飾品爆擊%(含強化)   // 基礎8%
爆擊傷害 = 1.6 倍(固定)
單次傷害 = 攻擊力 × (0.85 ~ 1.15 隨機) × (爆擊 ? 1.6 : 1)
技能傷害 = 單次 × 技能倍率 × (1 + 0.15×卡.xdmg)     // 絕技精通
```

### HP 上限 mhp
```
mhp = (60 + (劍士 +40) + lv×8 + 20×卡.hp + 裝備HP(防具+頭盔,含強化))
      × (1 + 0.08×永久.vit)          // 體魄強化
      × max(0.4, 1 - 0.15×血祭)      // 血祭代價
```

### MP 上限 mmp
```
mmp = 30 + (法師 +15) + lv×4 + 15×卡.mp      // 心靈之泉
```

### 減傷 armorDef
```
減傷 = 防具減傷 + 頭盔減傷 (皆含強化)   // 直接抵扣受到的傷害
```

### 移動 / 跳躍
```
移速 = (3.2 + 0.4×卡.spd + 鞋子移速(含強化) + (狂暴 ? 0.8 : 0)) × (凍結 ? 0.55 : 1)
跳躍力 = 11.5 + 鞋子跳躍加成
```

### 技能冷卻 slotCd
```
冷卻 = max(6, 技能基礎CD × 天賦冷卻 × 0.9^卡.aspd × (1 + 0.18×蠻力))
```

### 升級需求
```
下一級所需 EXP = 25 + lv×15
```

## 6. 掉落與經濟

```
裝備掉率 = min(8% + 1%×層 + 2%×永久.尋寶, 25%)   // 菁英必掉、Boss 保底史詩
藥水掉率 = 13% + 8%×卡.pot
靈魂獲取 = 場內靈魂 × (1 + 0.1×永久.靈魂共鳴) × (1 + 0.2×貪婪卡)
分解材料 = 白 石1 / 藍 石3塵1 / 黃 石6塵3 / 史 石10塵6 / 傳 石16塵10
```

## 7. 裝備品質(倍率)

| 品質 | 數值倍率 | 附魔槽(規劃) |
|---|---|---|
| 普通(白) | ×1.0 | 1 |
| 精良(藍) | ×1.5 | 1 |
| 稀有(黃) | ×2.1 | 2 |
| 史詩(紫) | ×2.8 | 2 |
| 傳說(橙) | ×3.6 | 3 |

裝備強化 +0~+12,每級 +5%(滿 +60%);+8 起失敗可能爆裝。

## 8. 怪物 / Boss 成長(參考)

```
怪物數值倍率 sc = (1 + 0.30×(層-1) + 0.02×(層-1)²) × (深淵 21+ 再 ×1.15)
經驗倍率 = 1 + 0.15×(層-1)
菁英機率 = min(8% + 2.5%×層, 40%)
Boss 血量 = 800×sc(每 5 層);傷害 = 15×sc;三階段、階段召援軍
```
