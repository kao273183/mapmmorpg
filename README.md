# mapmmorpg

瀏覽器像素 Roguelite RPG。

## 音效

首次按鍵或點擊後，遊戲會載入戰鬥、技能、拾取與介面音效。可在城鎮公告欄的設定視窗調整音量或靜音，設定會自動保存在瀏覽器。

音效素材與授權資訊請見 [`audio/LICENSE.md`](audio/LICENSE.md)。

## 戰鬥效果

戰鬥使用固定 60 Hz 更新，支援跳躍／技能輸入緩衝、全職業共通衝刺（`Shift`，2.5 秒冷卻）、分級攻擊數字、爆擊停頓、受傷提示與 Boss 落點預警。基礎移動速度為 2.0；觸控介面另有「衝」按鈕。法師火球會對前方飛行怪提供短時間的柔和對空修正。可在設定調整螢幕震動、戰鬥閃光、傷害數字與觸控震動。

手機版使用橫向戰鬥介面；直向開啟時會提示旋轉，支援全螢幕與瀏海安全區，畫面會以完整 16:9 比例縮放而不裁切 HUD。

## 地城路線

每章共 5 房。清除非 Boss 房後可在兩條公開危險與獎勵的路線間選擇，目前包含安全戰鬥、菁英、寶藏、事件、營地與群系險境；路線卡會顯示實際敵人、事件、地形及最低獎勵。同一地城種子與選擇可重現房型、平台、事件和敵人配置。五群系險境分別包含荊棘根鬚、落石、熔岩噴口、冰面與虛空平台；寶箱與祭壇則包含旅行者寶箱、寶箱怪、補給箱、封印寶箱、血之祝福、生命泉源、奧術泉源與命運祭壇。事件會避開最近結果，並在確認前公開成本與效果；菁英伏擊、限時殲滅、無傷試煉與地形試煉均可成功、失敗或安全拒絕，不會永久鎖住房間出口。擊敗第 5 房 Boss 後會依探索評價發放章節寶箱，玩家可返回基地或繼續深入下一章。

D2 地城內容已完整交付，並以 1,000 組種子、五種地形、十二種事件、桌機輸入、手機橫向觸控及死亡／撤退保存情境完成回歸。

遊戲設定可直接查看更新紀錄與 D3 平衡摘要。自然遊玩與固定基準各保留最近 60 局，包含路線選擇、房型耗時、承傷來源與撤退結果，並可複製為測試 JSON。

D3 固定基準提供劍士、法師各三組配對種子與固定裝備，可從設定直接開始。自然遊玩和基準測試分開統計；基準局結算後會還原角色進度。
