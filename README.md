# mapmmorpg

瀏覽器像素 Roguelite RPG。

## 音效

首次按鍵或點擊後，遊戲會載入戰鬥、技能、拾取與介面音效。可在城鎮公告欄的設定視窗調整音量或靜音，設定會自動保存在瀏覽器。

音效素材與授權資訊請見 [`audio/LICENSE.md`](audio/LICENSE.md)。

## 戰鬥效果

戰鬥使用固定 60 Hz 更新，支援跳躍／技能輸入緩衝、全職業共通衝刺（`Shift`，2.5 秒冷卻）、分級攻擊數字、爆擊停頓、受傷提示與 Boss 落點預警。基礎移動速度為 2.0；觸控介面另有「衝」按鈕。法師火球會對前方飛行怪提供短時間的柔和對空修正。可在設定調整螢幕震動、戰鬥閃光、傷害數字與觸控震動。
