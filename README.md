# mapmmorpg

瀏覽器像素 Roguelite RPG。

程式碼已集中於 `src/`；地城模組位於 `src/dungeon/`，主遊戲依責任拆分於 `src/game/`。遊戲直接載入與原始素材分別位於 `assets/runtime/`、`assets/source/`。目前的載入順序與放置規則請見 [專案結構](doc/PROJECT-STRUCTURE.md)。

## 音效

首次按鍵或點擊後，遊戲會載入戰鬥、技能、拾取與介面音效。可在城鎮公告欄的設定視窗調整音量或靜音，設定會自動保存在瀏覽器。

音效素材與授權資訊請見 [`assets/runtime/audio/LICENSE.md`](assets/runtime/audio/LICENSE.md)；素材分區與維護規則請見 [`assets/README.md`](assets/README.md)。

## 戰鬥效果

戰鬥使用固定 60 Hz 更新，支援跳躍／技能輸入緩衝、全職業共通衝刺（`Shift`，2.5 秒冷卻）、分級攻擊數字、爆擊停頓、受傷提示與 Boss 落點預警。基礎移動速度為 2.0；觸控介面另有「衝」按鈕。法師火球會對前方飛行怪提供短時間的柔和對空修正。可在設定調整螢幕震動、戰鬥閃光、傷害數字與觸控震動。

手機版使用橫向戰鬥介面；直向開啟時會提示旋轉，支援全螢幕與瀏海安全區，畫面會以完整 16:9 比例縮放而不裁切 HUD。

## 地城路線

每章共 5 房。清除非 Boss 房後可在兩條公開危險與獎勵的路線間選擇，目前包含安全戰鬥、菁英、寶藏、事件、營地與群系險境；路線卡會顯示實際敵人、事件、地形及最低獎勵。同一地城種子與選擇可重現房型、平台、事件和敵人配置。五群系險境分別包含荊棘根鬚、落石、熔岩噴口、冰面與虛空平台；寶箱與祭壇則包含旅行者寶箱、寶箱怪、補給箱、封印寶箱、血之祝福、生命泉源、奧術泉源與命運祭壇。事件會避開最近結果，並在確認前公開成本與效果；菁英伏擊、限時殲滅、無傷試煉與地形試煉均可成功、失敗或安全拒絕，不會永久鎖住房間出口。擊敗第 5 房 Boss 後會依探索評價發放章節寶箱，玩家可返回基地或繼續深入下一章。

D2 地城內容已完整交付，並以 1,000 組種子、五種地形、事件系統、桌機輸入、手機橫向觸控及死亡／撤退保存情境完成回歸。

遊戲設定可直接查看更新紀錄與 D3 平衡摘要。自然遊玩與固定基準各保留最近 60 局，包含路線選擇、房型耗時、承傷來源與撤退結果，並可複製為測試 JSON。

D3 固定基準提供劍士、法師各三組配對種子與固定裝備，可從設定直接開始。自然遊玩和基準測試分開統計；基準局結算後會還原角色進度。

D3-C 首輪依固定基準理論模型調整三項：菁英單體 HP -10.4%、落石／熔岩傷害 -12.5%、險境額外靈魂 +10%。設定中的平衡報表與完整 JSON 都會保留調整前後；自然遊玩樣本達門檻後再進行實測複核。

E1-A 已建立五群系 Boss 技術地基：五隻 Boss 使用獨立資料、共用階段／預警介面與固定測試入口；平衡報表可記錄擊殺時間、死亡招式及最終階段。完整改造順序請見 [Boss E1 計畫](doc/PLAN-boss-E1.md)。

E1-B 已完成翠綠草原 Boss：草原領主具備根鬚橫掃、殘留減速荊棘、種子彈幕與專屬樹冠像素外觀；第一階段維持單招教學，第二、三階段才逐步組合威脅。

E1-C 已完成幽暗洞窟 Boss：洞窟領主具備落石標記、雙向地面衝擊波與專屬岩石像素外觀；招式期間岩棚會標示為安全區，第二階段起才加入衝擊波組合。

E1-D 已完成熾熱熔岩 Boss：熔岩魔王具備完整路徑預警的熔岩衝鋒、依序啟動的連鎖噴發與專屬熔岩像素外觀；噴口熄火後會明確標示安全通行窗。

E1-E 已完成冰霜凍原 Boss：冰霜領主具備寒冰槍陣、留下冰面的暴風突進與專屬冰晶像素外觀；Boss 冰面沿用滑行慣性，玩家可持續反向輸入煞車。

E1-F 已完成虛空深淵 Boss：深淵魔王具備虛空彈幕、暫時消除浮空平台與專屬虛空像素外觀；地面主路徑永遠保留，第三階段也至少保留一座浮台。

E1-G 已完成五群系 Boss 平衡與收尾：五隻 Boss 各有劍士／法師配對基準、擊殺時間與承傷目標；設定新增 Boss 測試紀錄頁，配對時間差超過 15% 時提示複核。手機、低特效、死亡／撤退、Boss 掉落與舊存檔均納入完整回歸。

G1-A 已建立祝福與詛咒的每局技術地基：同一地城種子可重現選項，每局有兩次有限重抽並可安全拒絕；詛咒必須在選擇前同時揭露代價與對應收益。本批尚未加入實際效果或調整戰鬥數值，完整擴充順序請見 [事件 G1 計畫](doc/PLAN-events-G1.md)。

G1-B 已加入 12 種實際祝福：攻擊、防禦、機動與資源各 3 種，依前三章逐步解鎖。效果涵蓋傷害、技能、菁英與 Boss、最大生命、進房護盾、回復、移動、衝刺、跳躍、靈魂、裝備掉率及一次選卡重抽；每種祝福只能取得一次並有固定數值上限。

G1-C 已加入 12 種自願詛咒：戰鬥、奧術、生存與挑戰各 3 種，每種都在確認前同時公開代價與對應收益。效果涵蓋玩家、敵人、菁英、險境與 Boss；封印命運會將剩餘異變重抽換成兩次選卡重抽，最後燈火則以禁用復活換取高額靈魂。

G1-D 新增遺忘熔爐、流浪鍊金師、回音書庫、失落商隊與古代秘藏，事件定義增至 17 個、正式有效結果增至 20 種。新事件均可安全離開；動態交易會先顯示成本或收益，資源不足不扣款，完成後也不會重複發獎。

G1-E 已將祝福與詛咒接入正式探索：每隔 3～4 層交替出現三張候選卡，可使用每局兩次重抽或安全拒絕。桌機可按數字、`R`、`0`／`Esc` 操作，手機可直接點擊；HUD 的祝福／詛咒計數可用 `M` 或點擊開啟本局效果清單。

G1-F 已完成首輪平衡與收尾：沿用 D3 劍士／法師配對基準，比較無效果、只拿祝福、只拿詛咒與混合四種局況；本輪沒有警戒值超標，因此不做無樣本依據的數值調整。平衡紀錄現在會保存當局效果組合、房間獎勵、通關時間、承傷與靈魂，固定模型及判讀詳見 [G1 平衡報告](doc/REPORT-events-G1.md)。
