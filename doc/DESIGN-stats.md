# 能力素質表(玩家屬性與公式)

> 依目前 `src/game/` 內的系統程式整理。所有數值以程式碼為準,改動時同步更新本表。
> 符號:`lv`=角色等級、`cd.*`=升級卡層數、`meta.up.*`=永久強化等級、`enh`=裝備強化等級、`perkV()`=卡片效果、`affixV()`=已裝備附魔加總、部位數值皆已含強化倍率 `enhMul = 1 + 0.05×enh`。

## 1. 核心屬性公式

### 攻擊力 atkPow
```
base = 8 + lv×2.5 + 武器攻擊(含強化) + (劍士 +4)
倍率 = (1 + 0.12×卡.atk)        // 力量湧現
     × (1 + 0.04×永久.atk)      // 攻擊強化
     × (1 + 飾品攻擊%)          // 護符 atkMul(含強化)
     × (1 + 附魔.攻擊%)          // 鋒銳
     × (狂暴中 ? 1.3 : 1)       // 狂暴技能
     × (1 + 0.30×血祭) × (1 + 0.40×蠻力) × (1 + 0.45×玻璃大砲)
     × (HP≥95% ? 1 + 0.08×完美狀態 : 1)
     × (HP<35% ? (1 + 0.50×絕地反擊)×(1 + 附魔.嗜血狂) : 1)
攻擊力 = base × 倍率
```

### 傷害 / 爆擊
```
爆擊率 = 0.08 + 0.06×卡.crit + 0.005×永久.精準 + 飾品爆擊%(含強化) + 附魔.致命
爆擊傷害 = 1.6 + 附魔.狂虐
單次傷害 = 攻擊力 × (0.85 ~ 1.15 隨機) × (爆擊 ? 爆擊傷害 : 1)
技能傷害 = 單次 × 技能倍率 × (1 + 0.15×卡.xdmg) × (MP≥70% ? 1 + 0.1×奧術超載 : 1)
目標 HP<25% 時再乘 (1 + 0.1×處決者)
```

### HP 上限 mhp
```
mhp = (60 + (劍士 +40) + lv×8 + 20×卡.hp + 裝備HP(防具+頭盔,含強化))
      × (1 + 0.08×永久.vit)          // 體魄強化
      × (1 + 附魔.堅壁)
      × max(0.4, 1 - 0.15×血祭)      // 血祭代價
```

### MP 上限 mmp
```
mmp = 30 + (法師 +15) + lv×4 + 15×卡.mp      // 心靈之泉
```

### 減傷 armorDef
```
減傷 = 防具減傷 + 頭盔減傷 + 卡.鋼鐵皮膚 + 附魔.銅牆   // 直接抵扣受到的傷害
```

### 移動 / 跳躍
```
移速 = (2.0 + 0.4×卡.spd + 鞋子移速(含強化) + 附魔.迅捷 + (狂暴 ? 0.8 : 0)) × (凍結 ? 0.55 : 1)
跳躍力 = 11.5 + 鞋子跳躍加成
```

### 技能冷卻 slotCd
```
冷卻 = max(技能最低CD或6, 技能基礎CD × 天賦冷卻 × 0.9^卡.aspd × (1 + 0.18×蠻力) × max(0.35, 1 - 附魔.疾襲) × (1 - 0.015×永久.戰技熟練))

揮砍與火球術的基礎冷卻皆為 45 幀(0.75秒)，最低冷卻 18 幀(0.3秒)。
```

### 升級需求
```
下一級所需 EXP = round(50 + 30×lv + 5×lv²)
```

## 7. 掉落與經濟

```
一般裝備掉率 = min(2.5% + 0.25%×層 + 1%×永久.尋寶, 10%) + 附魔.尋寶
菁英裝備掉率 = 一般裝備掉率 + 15%(一般/菁英最終皆以 50% 為總上限)
Boss 裝備 = 保底 2 件(其中 1 件保底史詩,20 層後傳說)
藥水掉率 = 7% + 4%×卡.pot
一般怪靈魂掉率 = 25%(1 顆);菁英固定 2 顆;Boss 固定 8 顆
靈魂獲取 = 場內靈魂 × (1 + 0.05×永久.靈魂共鳴) × (1 + 0.1×貪婪卡) × (1 + 附魔.貪婪)
分解材料 = 白 石1 / 藍 石3塵1 / 黃 石6塵3 / 史 石10塵6 / 傳 石16塵10
```

### 地城章節獎勵（D1）

每 5 層為一個章節，前 4 個房間累積探索分數，擊敗第 5 層 Boss 後結算：

| 房型 | 探索分數 |
|---|---:|
| 安全／寶藏／事件 | 1 |
| 菁英 | 2 |
| 營地 | 0 |

| 章節探索分數 | Boss 寶箱 |
|---|---|
| 0～5 | 精良裝備 ×1、強化石 ×1 |
| 6 以上 | 稀有裝備 ×1、強化石 ×2 |

章節結算後可選擇「安全撤退」保留本局所得，或「繼續深入」進入下一章；繼續深入會重置該章探索分數。

## 9. 怪物 / Boss 成長(參考)

```
怪物數值倍率 sc = (1 + 0.30×(層-1) + 0.02×(層-1)²) × (深淵 21+ 再 ×1.15)
怪物耐久倍率 = 1.5 + min(0.75, 0.025×(層-1))
一般怪血量 = 怪物基礎血量 × sc × 怪物耐久倍率
經驗倍率 = 1 + 0.15×(層-1)
菁英機率 = min(8% + 2.5%×層, 40%)
Boss 血量 = 800×sc×1.35(每 5 層);傷害 = 15×sc;三階段、階段召援軍
第5層教學 Boss：血量倍率改為×1.08、傷害×0.82、攻擊前搖48幀、階段援軍1/2隻；第10層起維持標準公式。
```

## 10. 遊戲內能力面板

- 冒險或城鎮中按 `P` 開啟完整角色能力,再次按 `P` 或 `Esc` 關閉。
- 行動裝置可使用畫面上的 `P` 按鈕。
- 冒險中開啟能力面板會暫停戰鬥。
- 面板分為戰鬥、生存、成長／經濟三欄,數值直接取自實際公式。
- 攻擊、爆擊、HP、減傷、移速等項目會列出主要加成來源。
- 城鎮中顯示最近角色數值;開始新冒險後會依職業、裝備與卡片即時更新。
