# 技能系統改造設計書(M2)

> 版本 v1.1(2026-07-21)。狀態:**第一批功能與暗黑系技能介面已實作**。
> 核心:技能抽取(靈魂)+ 暗黑式分支天賦樹(流派、可重置)+ 出戰編輯(最多 3 招)。

## 1. 目標

- 技能收集感:抽取解鎖,取代寫死的 Z/X 兩招。
- 永久成長與流派:每招技能一棵分支天賦樹,二選一鎖定流派,花靈魂重置。
- Build 自由度:每場出發前編輯出戰技能,最多 3 招。
- 靈魂新消耗口:抽取(~960/職業)+ 重置(無上限),與永久強化(3,065)並列。

## 2. 技能庫

每職業 5 招 = 基礎 1 招(免費,預設解鎖)+ 抽取池 4 招。新手體驗不變。

| 職業 | 基礎(送) | 抽取池 |
|---|---|---|
| 劍士 | 揮砍 | 旋風斬、突進斬(位移+傷害)、震地波(前方地面衝擊)、狂暴(短時攻速+移速) |
| 法師 | 火球 | 落雷、冰錐(穿透+緩速)、隕石(大範圍高CD)、魔法盾(吸收傷害) |
| 弓箭手(後續) | 射箭 | 二連射、箭雨、後跳射(位移)、鷹眼(爆擊 buff) |

註:旋風斬/落雷由固定技能降級為抽取池技能。**不發補償抽**(已決定)。

## 3. 抽取

- 基地花 **40 靈魂/抽**,抽「當前所選職業」的池(不會抽到別職業)。
- 抽到新技能 → 解鎖;**抽到重複 → 該技能天賦點 +1**(上限 5,滿了不會再抽到該技能;池子全滿時停抽)。
- 無保底:池僅 4 招,期望值本身可控。
- 消耗估算:每職業全解鎖+天賦全滿 ≈ 4 × 6 × 40 = 960 靈魂。

## 4. 天賦:分支流派樹(每招一棵,5 點滿)

```
Lv1 強化 I(+12% 傷害)
Lv2 ★ 分支選擇(二選一,互斥,鎖定流派)
Lv3 分支強化(數值,隨分支不同)
Lv4 機制強化(冷卻/範圍等,隨分支不同)
Lv5 終極效果(分支大招)
```

點數沿樹依序投入;Lv2 投入時必須選 A/B 分支,之後 Lv3–5 走該分支。

### 流派範例

| 技能 | 流派 A | 流派 B |
|---|---|---|
| 火球 | 爆裂:濺射範圍傷害 → 終極:燃燒地面 | 連鎖:彈射 2/3 個敵人 → 終極:彈射傷害遞增 |
| 揮砍 | 重擊:高傷、目標 -1 → 終極:必爆擊 | 連擊:攻速快、目標 +2 → 終極:第 3 擊回 MP |
| 旋風斬 | 龍捲:時間延長、可移動施放 | 利刃:範圍小但傷害翻倍、擊殺重置冷卻 |
| 狂暴 | 血怒:HP 越低加成越高 | 戰意:擊殺延長持續 |

(其餘技能的分支在實作時逐一定義,遵循「A 面積/持續、B 單體/爆發」的大方向。)

### 重置

- 單技能重置:退回全部天賦點、解除分支,費用 `30 + 20 × 已投點數` 靈魂(滿 5 點 = 130)。
- UI 二段式確認(沿用背包分解模式)。

## 5. 出戰編輯

- 基地「技能」分頁採暗黑系石板風格，但所有圖示與介面均為本作程式繪製。
- 左側以圓形技能節點和路徑連線呈現技能樹；未解鎖顯示鎖頭，可投點與目前選取節點有光圈提示。
- 右側顯示技能圖示、MP／冷卻、說明、五階天賦路徑、流派選擇、升級與重置操作。
- 下方 3 個快捷槽對應 `Z / X / C`；選取技能後直接點槽位即可綁定、換位或卸下，仍保證至少保留一招。
- 戰鬥 HUD 同步改為圖示快捷列，施放冷卻時直接覆蓋倒數秒數。

## 6. 操作改版(已決定)

- 桌機:技能 `Z / X / C`,藥水改 **`A(紅)/ S(藍)`**。
- 手機:三顆技能鍵 + 兩顆藥水鍵,虛擬按鍵版面重排。

## 7. 戰鬥層重構

- `fireCd`/`boltCd` 等寫死冷卻 → `slotCd[0..2]` 通用陣列。
- 技能定義為 `{id, name, mp, cd, cast(slot, talent)}`;分支/天賦以參數傳入效果函式,不複製技能程式碼。
- 「絕技精通」卡改為「技能傷害 +10%(全部出戰技能)」。
- 重構完成後先原樣驗證舊兩招,再加新招。

## 8. 存檔 v2

- localStorage 與存檔碼追加:每招 `{天賦點數, 已投層數, 分支(0/1/未選)}` + 出戰配置。
- 匯入 v1 舊碼相容(技能欄位給預設值)。

## 9. 實作順序(3–4 次迭代)

1. 資料層:SKILL_DEFS、解鎖/天賦狀態、存檔 v2
2. 戰鬥重構:槽位通用化 + 舊兩招回歸驗證
3. 8 招新技能(先無分支特效的基礎版)
4. 天賦樹 UI + Lv2 分支選擇與數值差異(**第一批只做數值分支**)
5. 抽取/出戰 UI、鍵位與手機按鍵改版
6. Lv3–5 里程碑/終極特效分批補上、平衡調整

風險:戰鬥重構觸及施法動畫/MP/卡片加成所有路徑,是最易出 bug 的一步,獨立驗證後再前進。
