# 套裝系統設計

## 目標

- 套裝是深層地城的長期收集目標，不取代一般傳說裝與附魔流派。
- 每套固定四個部位：武器、防具、頭盔、鞋子；飾品維持自由搭配。
- 兩件提供流派方向，四件完成核心玩法；效果直接進入既有角色數值公式。
- 支援 2＋2 混搭，兩套的二件效果可同時啟動；不同套裝件數不會合併計算四件效果。

## 掉落與素材

套裝從第 5 層開始出現，生成後最低為史詩品質。

| 來源 | 每件裝備成為套裝的機率 |
|---|---:|
| 一般／菁英裝備 | 2% |
| 寶箱／試煉 | 5% |
| Boss 裝備 | 8% |

- 僅掉落目前職業的套裝。
- 分解史詩套裝取得套裝核心 ×2；傳說套裝取得 ×3。
- 每週活躍 220 額外取得套裝核心 ×1。
- 消耗套裝核心 ×6，可從已選套裝鍛造一件尚未收藏的部位；四件皆有時隨機鍛造一件。

## 存檔相容

- 套裝 ID 儲存在裝備物件的 `setId`。
- 套裝核心儲存在 `meta.mats.set`；舊存檔缺少欄位時補為 0。
- 無效套裝 ID 在載入時移除，不影響原裝備數值與附魔。
