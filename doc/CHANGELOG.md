# 開發完成紀錄

## v0.29.0 — E1-A 五群系 Boss 技術地基（2026-07-22）

- 新增獨立 Boss 模組，集中定義草原、洞窟、熔岩、冰霜與虛空五隻 Boss 的身分、配色、場地、階段與環境資料。

## v0.28.9 — D3-C 首輪數值校準（2026-07-22）

- 以 D3-B 六組固定基準的理論模型進行首輪校準；自然遊玩樣本尚未達門檻，因此不宣稱玩家實測已驗證。
- 菁英單體 HP 係數由 2.40 降至 2.15（-10.4%），降低菁英房單一目標拖時，但不改敵人數與攻擊力。
- 落石與熔岩噴口由最大 HP 8% 降至 7%（-12.5%）；荊棘、冰面與虛空平台維持原值。
- 險境完成的額外靈魂倍率提高 10%；第二章起固定由 10 提高為 11，強化石獎勵維持不變。
- 設定的 D3 平衡報表顯示校準摘要，完整 JSON 保留三項調整前後、幅度、依據與目標。
- 新增 D3-C smoke test，鎖定最多三項、每項 10～15%、實際接線、報表與版本資源。

## v0.28.8 — D3-B 固定基準局與報表（2026-07-22）

## v0.28.5 — 地城 D2-E 四種試煉（2026-07-22）

- 建立統一 `floorTrial` 狀態機，集中管理 offered、active、success、failed、declined 與單次獎勵狀態。
- 菁英伏擊改由共用試煉流程生成 3 名菁英守衛；成功後取得稀有裝備、附魔塵與額外探索評價。

- 試煉 HUD 顯示倒數、波次、擊敗數、無傷與地形通過狀態；首次地形教學會自動避開 HUD。
- 新增 D2-E smoke test，驗證四試煉抽選、狀態轉移、成功／失敗／拒絕、房門解除及獎勵防重複。
