# 地城 D3 執行計畫：平衡與測試紀錄

> 狀態：D3-A 本機紀錄與遊戲內摘要已實作；D3-B 數據校準、D3-C 數值調整待累積樣本。建立日期：2026-07-22。

## 1. 平衡原則

D3 不以單場體感直接改數值。每個職業至少累積 30 局，先確認問題屬於「資訊不清、風險過高、獎勵不足、房間過長或職業落差」，再一次只調一組變數，幅度先控制在 10～15%。調整後另收至少 20 局比較，不用同一批資料同時提出問題與證明修正有效。

## 2. 已記錄資料

每局只在玩家瀏覽器保存，最多保留最近 60 局，足以讓劍士與法師各累積 30 局：

- 職業、最高樓層、擊殺、靈魂、整局時間、死亡或撤退。
- 各房型出現在路線卡的次數與實際選擇次數。
- 各房型進入、完成與完成耗時。
- 實際扣除 HP 的承傷來源與總量；護盾完全吸收不列入承傷。
- 四種試煉的成功、失敗與拒絕次數。

設定中的「D3 平衡紀錄」顯示摘要，並可複製完整 JSON 供測試比較。資料不會上傳，也不寫入角色存檔碼。

## 3. 首輪目標區間

| 指標 | 目標 | 超出區間時優先檢查 |
|---|---:|---|
| 每章時間 | 8～12 分鐘 | 敵量、世界寬度、事件閱讀時間 |
| 一般單局時間 | 12～20 分鐘 | 補給頻率、撤退節點、房間長度 |
| 高風險路線選擇率 | 35～65% | 低於 35% 先檢查獎勵與預覽；高於 65% 檢查風險是否不足 |
| 撤退率 | 25～55% | 過低檢查撤退價值；過高檢查續戰獎勵與下一章難度 |
| 菁英房耗時／安全房 | 1.2～1.5 倍 | 怪物血量、菁英數量、最低獎勵 |
| 險境房耗時／安全房 | 1.0～1.3 倍 | 地形等待時間、安全路徑與材料獎勵 |
| 單一承傷來源占比 | 不高於 35% | 預警辨識度、命中區、傷害或出現頻率 |
| 劍士／法師平均樓層差 | 不超過 1.5 層 | 對空、近戰貼身風險、MP／補給效率 |

這些是首輪警戒線，不是硬性成功率。玩家主動選擇高風險流派時，承傷與死亡較高可以接受；但獎勵必須同步提高。

## 4. 判讀與調整順序

1. 先看路線是否出現：沒有被提供的房型不能用選擇率判斷。
2. 再看選擇率：低選擇但完成率正常，優先改善預覽或獎勵。
3. 再看耗時與承傷：耗時高但承傷低通常是等待或血量問題；承傷高但耗時正常通常是預警或傷害問題。
4. 最後看死亡／撤退與職業差：只有跨至少 30 局、兩職業都出現的差異才改全域數值。
5. 每輪只調一層：獎勵、敵量、地形傷害、補給或職業係數擇一。

## 5. 批次

### D3-A：本機紀錄與遊戲內摘要（已完成）

- 建立最近 60 局的本機平衡資料。
- 接入路線、房間、承傷、試煉與結算生命週期。
- 設定加入摘要與 JSON 複製入口。
- 設定加入可翻閱的遊戲內更新紀錄。

### D3-B：固定基準局與報表（下一批）

- 劍士、法師各建立三組固定裝備與種子。
- 自動彙整房型出現率、選擇率、完成時間與承傷占比。
- 分開自然遊玩資料與固定基準測試，避免樣本混用。

### D3-C：首輪數值校準

- 依 D3-B 結果選出最多三個超標指標。
- 每項以 10～15% 小幅調整，留下調整前後比較。
- D1、D2 全部回歸通過後再封版。
